# 弱いパスワードポリシー

## 対象ラボ

### 1. 脆弱なパスワードの登録

- **概要**: `123456` や `password` 等の脆弱なパスワードが登録可能
- **攻撃例**: 弱いパスワードで登録 → ブルートフォースで即突破
- **技術スタック**: Hono API (バリデーション)
- **難易度**: ★☆☆
